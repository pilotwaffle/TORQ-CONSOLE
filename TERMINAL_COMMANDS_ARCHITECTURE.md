## Security Layer Details

### Layer 1: Command Parsing

### Layer 2: Blocked Commands
```python
if base_command in BLOCKED_COMMANDS:
    raise DangerousCommandError(...)
# Blocks: rm, sudo, curl, chmod, etc.
```

### Layer 3: Whitelist Validation
```python
if base_command not in SAFE_COMMANDS:
    raise CommandNotWhitelistedError(...)

# Check subcommands
if SAFE_COMMANDS[base_command] is not None:
    if subcommand not in SAFE_COMMANDS[base_command]:
        raise SecurityViolationError(...)
```

### Layer 4: Input Sanitization
```python
DANGEROUS_CHARS = {';', '|', '&', '>', '<', '`', '$', '\n', '\r'}

for dangerous_char in DANGEROUS_CHARS:
    if dangerous_char in command:
        raise SecurityViolationError(...)
```

### Layer 5: Working Directory Validation
```python
def _validate_working_directory(self, working_dir: str):
    path = Path(working_dir).resolve()

    if not path.exists():
        raise InvalidWorkingDirectoryError(...)

    if not path.is_dir():
        raise InvalidWorkingDirectoryError(...)

    for restricted in RESTRICTED_DIRS:
        if str(path).startswith(restricted):
            raise InvalidWorkingDirectoryError(...)
```

### Layer 6: Timeout Enforcement
```python
if timeout is None:
    timeout = DEFAULT_TIMEOUT  # 30s

if timeout > MAX_TIMEOUT:
    timeout = MAX_TIMEOUT  # 300s cap

subprocess.run(..., timeout=timeout)
```

### Layer 7: Secure Execution
```python
subprocess.run(

    cwd=working_dir,
    shell=False,        # CRITICAL: Never True
)
```
